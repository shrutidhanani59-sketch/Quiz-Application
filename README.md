# 🏅 Quiz-Application 

This is a <b>Quiz Application </b> ,  Making HTML , CSS , JavaScript . Next , Previous  and Submit button also add . 

## 🚀 Features

📚 Multiple-choice questions
⏱️ Countdown timer for each question
✅ One answer selection per question
➡️ Next/Submit button 
📊 Final score display
🔄 Restart quiz option
🎨 Clean and modern UI


## 🛠️ Technologies Used
-HTML
-CSS
-JavaScript
